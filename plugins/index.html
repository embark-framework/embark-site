Plugins

Embark supports plugins to extend or enhance its own functionality or even to support other technologies in its stack.
Please refer to the documentation on how to install a plugin or how to easily create one!

  • embark-bamboo

    Plugin to add Bamboo support to embark

    language smart-contracts
  • embark-solc

    Uses the command line Solidity Compiler for light-speed compile times

    solidity language smart-contracts
  • embark-solium

    The official Solium Plugin for the Embark Framework

    linter solidity solium
  • embark-status

    Status plugin to interact with embark

    status mobile
  • embark-remix

    Plugin to start Remix and Remix directly with Embark

    remix debugger
  • embark-fortune

    Displays a fortune message after every build

    fun

Legacy

These plugins are meant for Embark 3.1 and no longer necessary due to the new pipeline functionality in Embark 3.2.
Please see the new pipeline documentation here for more information

  • embark-pug

    Embark plugin to support pug / jade templates

    pug jade templates
  • embark-haml

    Embark plugin to support haml templates

    haml templates
