# Embark Official Website

The website for Embark.

## Getting started

Install dependencies:

``` bash
$ git clone https://github.com/status-im/embark-site.git
$ cd embark-site
$ npm install
```

Generate:

``` bash
$ hexo generate
```

Run server:

``` bash
$ hexo server
```
